# Resumon

**Resumon** is a web-based platform that helps students, job seekers, and professionals create professional, ATS-ready resumes with customizable templates and real-time preview.

---

## Features

- User registration and profile management
- Step-by-step guided forms for personal, educational, professional, and project information
- Multiple customizable resume templates (modern, classic, creative, ATS-friendly)
- Real-time resume preview
- Export resumes as PDF or DOCX
- Optional AI-powered data input (Gemini API)
- Responsive UI for desktop, tablet, and mobile
- Cloud storage to save and edit resumes anytime
